# シンプルなボックスに変身する機能

## 🎯 目標

複雑なヘビのモデルではなく、シンプルなボックス（Part）に変身する機能から始めます。
これにより、基本的な変身機能を確認してから、段階的に複雑なモデルに移行できます。

## 📋 手順

### ステップ1: Roblox StudioでシンプルなPartを作成

1. **ServerStorageを開く**
   - ExplorerパネルでServerStorageを確認

2. **Partを作成**
   - ServerStorageを右クリック
   - `Insert Object > Part` を選択
   - または、`Insert > Object > Part` を選択

3. **Partの名前を変更**
   - 作成されたPartを選択
   - Propertiesパネルで「Name」を「SnakeModel」に変更
   - **重要**: コードが「SnakeModel」という名前を探すため、正確にこの名前にする

4. **Partの設定**
   - Partを選択
   - Propertiesパネルで以下を設定：
     - **Size**: (2, 2, 4) - ヘビのような細長い形状
     - **Color**: 緑系の色（例: 緑、濃い緑）
     - **Material**: 必要に応じて変更（Plastic、SmoothPlasticなど）

### ステップ2: コードの確認

現在のコードは既にPartに対応しています。以下の処理で動作します：

1. ServerStorageから「SnakeModel」を取得
2. モデル内のBasePart（今回の場合はPart）を検出
3. プレイヤーの位置に配置
4. サイズを調整

### ステップ3: 動作確認

1. **ファイルを保存**
   - 変更がRojoで自動的に同期されることを確認

2. **ゲームを実行**
   - 「Play」ボタンをクリック
   - Outputウィンドウでエラーがないことを確認

3. **プレイヤーを確認**
   - プレイヤーが緑のボックスに変身していることを確認
   - サイズが正しく設定されていることを確認

## 🎨 カスタマイズ（オプション）

### よりヘビらしい見た目にする

1. **Sizeを調整**
   - より細長く: (1, 1, 6)
   - より太く: (3, 3, 4)

2. **Colorを変更**
   - 緑系: Color3.fromRGB(0, 150, 0)
   - 毒々しい色: Color3.fromRGB(100, 200, 100)

3. **Materialを変更**
   - SmoothPlastic: 滑らかな見た目
   - Neon: 光る見た目

### 複数のPartで構成する

1. **Modelを作成**
   - ServerStorageを右クリック
   - `Insert Object > Model` を選択
   - 名前を「SnakeModel」に変更

2. **Partを追加**
   - Model内に複数のPartを追加
   - ヘビの体のように配置

3. **各Partを設定**
   - サイズ、色、位置を調整

## 🔄 次のステップ

シンプルなボックスが正しく動作することを確認したら：

1. ✅ 基本的な変身機能が動作する
2. ✅ サイズ調整が正しく動作する
3. ⏳ より複雑なモデル（ヘビなど）に移行

## 💡 メリット

シンプルなボックスから始めるメリット：

1. **問題の特定が容易**
   - 複雑なモデルの問題を除外できる
   - 基本的な機能に集中できる

2. **デバッグが簡単**
   - Partはシンプルで問題が起きにくい
   - 動作確認が容易

3. **段階的な開発**
   - 基本機能を確認してから複雑化
   - リスクを最小化
